## Key Technical Details

### AUKF Target Specifications
- Position accuracy: <0.5m (RTK), <2m (GNSS)
- Attitude accuracy: <0.5°
- Runtime: <1ms per update

### LAMBDA Integer Ambiguity Resolution
- Used for RTK GPS positioning
- Requires 4+ satellites with fixed ambiguities

### Safety Features
- Battery thresholds: 10% (land), 20% (RTH)
- GPS: 8+ satellites required
- Geofence: max 100m altitude, 500m from home
