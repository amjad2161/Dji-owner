## חלק ד': נוסחאות ואלגוריתמים

### ✅ Kalman Filter - מיושם מלא
```
x̂_{k|k-1} = F_k · x̂_{k-1|k-1} + B_k · u_{k-1}
P_{k|k-1} = F_k · P_{k-1|k-1} · F_k^T + Q_k
K_k = P_{k|k-1} · H_k^T · (H_k · P_{k|k-1} · H_k^T + R_k)^{-1}
```

### ✅ EKF - מיושם מלא
```
F_k = ∂f/∂x |_{x̂_{k-1|k-1}}
H_k = ∂h/∂x |_{x̂_{k|k-1}}
```

### ✅ UKF - מיושם מלא
```
ℳ_i = x̂ + γ·√(LP)_i
x̂_k|k-1 = Σ W_i^m · ℳ_k|k-1
```

### ✅ AUKF - מיושם מלא
```
21-25 state vector
α = 0.001, κ = 0, β = 2
Square-root implementation
```
