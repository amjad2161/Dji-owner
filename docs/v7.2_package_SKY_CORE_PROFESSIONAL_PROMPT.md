# פרומפט מקיף ומקצועי — מערכת SkyCore לשליטה מלאה ברחפנים

## מזהה המשימה
בניית מערכת שליטה מקצועית מלאה לרחפנים — **SkyCore Drone Operating System** — עם כל היכולות, השכבות, הנוסחאות, והקוד הנדרש.

---

## 1. סקירה כללית של הפרויקט

### 1.1 מה זה SkyCore
SkyCore היא פלטפורמת שליטה מקצועית ומלאה ברחפנים הכוללת:
- **Ground Control Station (GCS)** — ממשק ותוכנה לשליטה ברחפנים
- **Unified Drone Runtime** — ממשק אחיד לכל סוגי הרחפנים
- **Counter-UAS (C-UAS)** — זיהוי ומעקב אחרי רחפנים לא מורשים
- **Navigation & Estimation** — ניווט מדויק עם GNSS/RTK ומסנני Kalman
- **Autonomy** — אוטונומיה מלאה כולל SLAM, זיהוי אובייקטים, תכנון מסלול

### 1.2 היקף הפרויקט
- **~359 קבצי Python** (79+ מודולים)
- **~120,000+ שורות קוד**
- **GCS WebApp** — React/TypeScript
- **מסמכים טכניים** — Kalman Filter, EKF, UKF, AUKF

### 1.3 דרישות מהותיות
```
✅ הכל חייב להיות אמיתי — ללא סימולציה או דימוי
```

### שכבה 3: State Estimation — AUKF (Augmented Unscented Kalman Filter)
**תיאור:** ליבת הניווט המדויק — החשובה ביותר

#### 3.1 State Vector (21-25 states)
```
x = [p_N, v_N, q, b_g, b_a, w_N, c_b, c_b_dot, ...]^T
```
- `p_N` = Position in NED (3D)
- `v_N` = Velocity in NED (3D)
- `q` = Quaternion attitude (4D)
- `b_g` = Gyro bias (3D)
- `b_a` = Accelerometer bias (3D)
- `w_N` = Wind velocity (3D)
- `c_b` = Clock bias (scalar)
- `c_b_dot` = Clock drift (scalar)

#### 3.2 Process Model
```
p_dot = v

v_dot = C_b^N (a - b_a) + g + w

q_dot = 0.5 · q ⊗ (ω - b_g)

b_g_dot = w_bg

b_a_dot = w_ba

w_dot = -w/τ_w + w_w

c_b_dot = c_b_dot

c_b_ddot = w_clk
```

#### 3.3 Measurement Models

**GNSS Pseudorange:**
```
ρ_i = ||p_E - p_sat_i|| + c_b + I_i + T_i + ε
```

**GNSS Carrier Phase (RTK):**
```
φ_i = ||p_E - p_sat_i|| + c_b + I_i + T_i + λ·N_i + ε
```
עם:
- Klobuchar ionospheric model
- Saastamoinen tropospheric model
- Dual-frequency iono-free combination
- LAMBDA ambiguity resolution (full integer least-squares)

**IMU, Baro, Mag, Vision measurements**

#### 3.4 AUKF Parameters
```
α = 0.001
κ = 0
β = 2
L = 21 (state dimension)
λ = α²(L + κ) - L
```

#### 3.5 Sigma Points
```
ℳ_0 = x̂
ℳ_i = x̂ + √((L+λ)P)_i         for i = 1..L
ℳ_i = x̂ - √((L+λ)P)_i         for i = L+1..2L
```

#### 3.6 Weights
```
W_0^m = λ/(L+λ)
W_0^c = W_0^m + (1 - α² + β)
W_i^m = W_i^c = 1/(2(L+λ))     for i = 1..2L
```

#### 3.7 Prediction Step
```
ℳ_k|k-1 = f(ℳ_k-1|k-1, u_IMU, Δt)
x̂_k|k-1 = Σ W_i^m · ℳ_k|k-1
P_k|k-1 = Σ W_i^c (ℳ_k|k-1 - x̂)(·)^T + Q_k
```

#### 3.8 Update Step (Asynchronous, Adaptive)
```
𝒵 = h(ℳ_k|k-1)
ẑ = Σ W_i^m · 𝒵
S = Σ W_i^c (𝒵 - ẑ)(·)^T + R_k
P_xz = Σ W_i^c (ℳ_k|k-1 - x̂)(𝒵 - ẑ)^T
K = P_xz · S⁻¹
x̂_k|k = x̂_k|k-1 + K(z - ẑ)
P_k|k = (I - KH_eff) P_k|k-1 (I - KH_eff)^T + KRK^T
```

#### 3.9 Adaptive Tuning
- **R Adaptive:** Multipath detection via innovation whiteness test
- **Q Adaptive:** Vibration detection via IMU PSD analysis
- **Square-Root UKF:** Cholesky decomposition + QR for numerical stability

#### 3.10 Performance Requirements
```
Position RMSE:
  - RTK: < 0.5m
  - GNSS with outages: < 2m
Attitude accuracy: < 0.5°
Wind estimation: < 0.4 m/s
Runtime: < 1ms per update
```

### שכבה 4: Flight Control
**תיאור:** בקרת טיסה — Attitude, Position, Trajectory

#### 4.1 Inner Loop — Geometric Quaternion Control + Feedforward
```
τ = -k_p · e_q - k_d · ω_e + ω × Jω + J · ω̇_d
```
כאשר:
- `e_q` = Quaternion error
- `ω_e` = Angular velocity error
- `J` = Inertia matrix
- `k_p, k_d` = Gains

#### 4.2 Outer Loop — Cascaded PID/LQR/MPC
```
a_cmd = -K_p(p - p_d) - K_d(v - v_d) + v̇_d + g - w_est
```
עם wind compensation.

#### 4.3 Model Predictive Control (MPC)
- 20-step horizon
- Constraints on thrust and angles
- Cost function J minimization
- QP solver integration

#### 4.4 Motor Allocation Matrix

## 5. נוסחאות מתמטיות מלאות

### 5.1 Kalman Filter (Linear)
**Process Model:**
```
x_k = F_k · x_{k-1} + B_k · u_{k-1} + w_{k-1}
z_k = H_k · x_k + v_k
```

**Prediction:**
```
x̂_{k|k-1} = F_k · x̂_{k-1|k-1} + B_k · u_{k-1}
P_{k|k-1} = F_k · P_{k-1|k-1} · F_k^T + Q_k
```

**Update:**
```
K_k = P_{k|k-1} · H_k^T · (H_k · P_{k|k-1} · H_k^T + R_k)^{-1}
x̂_{k|k} = x̂_{k|k-1} + K_k · (z_k - H_k · x̂_{k|k-1})
P_{k|k} = (I - K_k · H_k) · P_{k|k-1}
```

### 5.2 Extended Kalman Filter (Nonlinear)
```
x_k = f(x_{k-1}, u_k) + w_k
z_k = h(x_k) + v_k
```

**Jacobians:**
```
F_k = ∂f/∂x |_{x̂_{k-1|k-1}}
H_k = ∂h/∂x |_{x̂_{k|k-1}}
```

**Same prediction/update structure with F_k, H_k**

### 5.3 Unscented Kalman Filter
**Sigma Points Generation:**
```
λ = α²(L + κ) - L
ℳ_0 = x̂
ℳ_i = x̂ + γ·√(LP)_i         for i = 1..L
ℳ_i = x̂ - γ·√(LP)_i         for i = L+1..2L
γ = √(L + λ)
```

**Prediction (UT):**
```
ℳ_k|k-1 = f(ℳ_k-1|k-1, u_k)
x̂_k|k-1 = Σ W_i^m · ℳ_k|k-1
P_k|k-1 = Σ W_i^c (ℳ_k|k-1 - x̂)(·)^T + Q_k
```

**Update (UT):**
```
𝒴 = h(ℳ_k|k-1)
ŷ = Σ W_i^m · 𝒴
S = Σ W_i^c (𝒴 - ŷ)(·)^T + R_k
P_xy = Σ W_i^c (ℳ_k|k-1 - x̂)(𝒴 - ŷ)^T
K = P_xy · S⁻¹
x̂_k|k = x̂_k|k-1 + K(y - ŷ)
P_k|k = P_k|k-1 - K · S · K^T
```

The DJI module provides comprehensive optimization for DJI Mavic, Mini, Air, and other DJI series drones within legal and manufacturer constraints.

```python
# Check for updates
```

### 12.7 Usage Example

```python
from skycore.dji import (
    create_dji_integration,
    OptimizationLevel,
    DJIIntegrationHub
)

# Create integration
dji = create_dji_integration(
    drone_model="mavic_3",
    optimization_level="ENHANCED"
)

# Connect
await dji.connect()

# Get full status
status = await dji.get_status()
print(f"Battery: {status['telemetry'].battery_percent}%")
print(f"Mode: {status['telemetry'].flight_mode}")

# Apply optimization
result = await dji.optimize(OptimizationLevel.PROFESSIONAL)

# Execute flight mode
await dji.controller.execute_intelligent_flight_mode(
    mode="Sport",
    params={}
)

# Disconnect
await dji.disconnect()
```
