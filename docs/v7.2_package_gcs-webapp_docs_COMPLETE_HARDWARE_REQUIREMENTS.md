# SkyCore Pro - רשימת חומרה מלאה ומקיפה

## סקירה כללית

מערכת SkyCore Pro מורכבת מ-6 שירותים עיקריים שלכל אחד יש דרישות חומרה ייחודיות:

| שירות | מטרה | חומרה נדרשת |
|-------|------|-------------|
| C-UAS Defense | הגנה מאיומים | RTL-SDR, LNA |
| Frequency Scanner | סריקת תדרים | RTL-SDR, אנטנה רב-תדרית |
| SDK Integration | חיבור רחפנים | DJI OcuSync / Autel SkyLink |
| ADS-B Service | מעקב מטוסים | RTL-SDR, אנטנת 1090MHz |
| GNSS Service | ניווט מדויק | u-blox RTK |
| Server | הרצת המערכת | מחשב חזק |

## 🔴 RTL-SDR Blog V3 ($30-35)

### למה נדרש?
```
┌─────────────────────────────────────────────────────────┐
│                    RTL-SDR V3                          │
├─────────────────────────────────────────────────────────┤
│  1. C-UAS Defense: קליטת אותות RF מרחפנים              │
│  2. Frequency Scanner: סריקת תדרים 500kHz-1.7GHz       │
│  3. ADS-B: קליטת מידע ממטוסים (1090MHz)               │
└─────────────────────────────────────────────────────────┘
```

### מפרט טכני:
- טווח תדרים: 500kHz - 1.7GHz
- רזולוציה: 8-bit ADC (למטוסים), 12-bit (ל-RF scanning)
- תמיכה: Windows, Linux, Raspberry Pi
- USB 2.0

### כמה צריך?
```
מינימום: 2 יחידות (אחת ל-ADS-B, אחת ל-RF Scanner)
מומלץ: 3 יחידות (לסריקה בו-זמנית)
```

### קישורים:
- [RTL-SDR.com](https://www.rtl-sdr.com)

## ⚠️ הערות חשובות

### חוקיות:
- **RTL-SDR**: חוקי לקליטה בלבד ברוב המדינות
- **Jamming**: אסור לפי חוק בישראל (ללא רשיון מתאים)
- **RTK**: דורש רשיון מנתונים בישראל

### תאימות:
- **DJI OcuSync**: עובד רק עם רחפני DJI
- **Autel SkyLink**: עובד רק עם רחפני Autel
- **RTL-SDR**: עובד עם הכל (קליטה)

### מה אפשר לדלג עליו בהתחלה:
```
דילוג אופציונלי:
□ Reach RS3 Base Station - אפשר להתחיל בלי RTK base
□ UPS - אפשר להתחיל בלי
□ Autel SkyLink - רק אם יש רחפן Autel
```
