## ⚠️ הערות חשובות

### רגולציה (חשוב!)
- **RTL-SDR:** חוקי לקליטה בלבד
- **HackRF:** דורש רשיון לשידור בישראל
- **תדרים:** בדוק את תקנות משרד התקשורת
- **RTK:** דורש רשיון מנתונים

### תאימות
- **DJI OcuSync:** עובד רק עם רחפני DJI
- **RTL-SDR:** עובד עם כל רחפן (קליטת RF)
- **Autel SkyLink:** דורש SDK ייעודי

### התקנה
- **Raspberry Pi:** התקן Raspberry Pi OS
- **NUC:** התקן Ubuntu 22.04 LTS
- **Docker:** מומלץ להרצת services
