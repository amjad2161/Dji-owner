## Key Features

### State Estimation (AUKF)
- **22-State Augmented Unscented Kalman Filter**
- Parameters: Оұ=0.001, Оә=0, ОІ=2
- States: position(3), velocity(3), quaternion(4), gyro_bias(3), accel_bias(3), wind(3), clock(2), misc(1)

### Flight Control
- **SE(3) Geometric Controller** with quaternion-based attitude control
- **Cascaded Position вҶ’ Velocity вҶ’ Attitude вҶ’ Motor** architecture
- **LQR Controller** with gain scheduling
- **Minimum Snap Trajectory Generation** for smooth paths

### Navigation & Planning
- **A*** optimal path planning with 3D occupancy grid
- **RRT*** informed sampling-based planning
- **Velocity Obstacle (VO)** reactive collision avoidance
- **ORCA** optimal reciprocal collision avoidance for swarms

### Communication
- **MAVLink v2** protocol implementation
- **ExpressLRS** long-range telemetry (420kbaud)
- **Iridium SBD** satellite communication
- **Starlink** high-bandwidth integration

### Perception & Autonomy
- **VIO** Visual-Inertial Odometry with MSCKF
- **YOLOv8** real-time object detection with TensorRT
- **BoT-SORT** multi-object tracking
- **C-UAS** counter-drone capabilities

### Digital Twin & GCS
- Real-time 3D visualization
- 7-page Hebrew RTL web interface
- Voice control (Hebrew)
- WebSocket real-time updates

## Usage

```python
from skycore import *
from skycore.adapters.pixhawk import PixhawkAdapter
from skycore.core.types import FlightMode

# Create adapter
adapter = PixhawkAdapter()

# Connect
await adapter.connect()

# Arm and takeoff
await adapter.arm()
await adapter.takeoff(altitude=50)

# Navigate to position
await adapter.goto_position(
    lat=32.0853,
    lon=34.7812,
    alt=50,
    yaw_deg=180
)

# Return to launch
await adapter.rtl()
```

## Technical Specifications

### AUKF Parameters
- **Оұ** (spread): 0.001
- **Оә** (secondary): 0.0
- **ОІ** (prior knowledge): 2.0 (Gaussian)
- **О»**: ОұВІ(n+Оә)-n = 0.001ВІ(22+0)-22 = -21.999978

### State Vector (22 dimensions)
```
x = [position_ned(3), velocity_ned(3), quaternion(4),
     gyro_bias(3), accel_bias(3), wind_ned(3),
     clock_bias(1), clock_drift(1), misc(2)]
```

### Control Limits
- Max tilt: 45В°
- Max yaw rate: 180В°/s
- Max vertical speed: 5 m/s
- Max horizontal speed: 15 m/s
