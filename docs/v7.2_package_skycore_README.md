## ארכיטקטורה

SkyCore Pro מיושמת בארכיטקטורת 8 שכבות:

```
┌─────────────────────────────────────────────────┐
│              Layer 8: GCS (React + WebSocket)    │
├─────────────────────────────────────────────────┤
│            Layer 7: Autonomy (VIO/YOLO)          │
├─────────────────────────────────────────────────┤
│      Layer 6: Communication (MAVLink/IRIDIUM)     │
├─────────────────────────────────────────────────┤
│         Layer 5: Navigation (A*/RRT*/VO)          │
├─────────────────────────────────────────────────┤
│       Layer 4: Control (Geometric/MPC/PID)       │
├─────────────────────────────────────────────────┤
│            Layer 3: AUKF State Estimation        │
├─────────────────────────────────────────────────┤
│              Layer 2: Sensors (IMU/GNSS)        │
├─────────────────────────────────────────────────┤
│          Layer 1: HAL (ICM-42688 + ZED-F9P)      │
└─────────────────────────────────────────────────┘
```

## תכונות עיקריות

### 🎯 אומדן מצב (State Estimation)
- **AUKF** - Augmented Unscented Kalman Filter עם 22 משתני מצב
- פרמטרים: α=0.001, κ=0, β=2
- אומדן אדפטיבי של רעש תהליך ומדידה

### 🛸 בקרת טיסה
- **Geometric Controller** - בקרה גיאומטרית SE(3)
- **Cascaded Controller** - מיקום → מהירות → זווית → מנועים
- **LQR Controller** - בקר לינארי ריבועי
- **Trajectory Generator** - יצירת מסלולים עם Minimum Snap

### 🗺️ ניווט ותכנון
- **A*** - תכנון מסלולים אופטימליים בסביבות מוכרות
- **RRT*** - תכנון מבוסס-דגימה לסביבות מורכבות
- **Velocity Obstacle (VO)** - הימנעות מהתנגשות

### 📡 תקשורת
- **MAVLink v2** - פרוטוקול תקשורת עיקרי
- **ExpressLRS** - טלמטריה ארוכת-טווח
- **Iridium SBD** - לווייני
- **Starlink** - רוחב פס גבוה

### 👁️ ראייה ממוחשבת
- **VIO** - Visual-Inertial Odometry
- **YOLOv8** - זיהוי עצמים בזמן אמת
- **BoT-SORT** - מעקב אחר עצמים מרובים

### 🛡️ C-UAS (Counter-UAS)
- זיהוי רחפנים עוינים
- מעקב וסיווג איומים
- הימנעות אוטונומית

## התקנה

```bash
# התקנת חבילות
pip install skycore

# או התקנה מפיתוח
cd skycore
pip install -e ".[dev]"
```

## שימוש בסיסי

```python
from skycore import *
from skycore.adapters.pixhawk import PixhawkAdapter

# יצירת מתאם
adapter = PixhawkAdapter()

# התחברות
await adapter.connect()

# המראה
await adapter.takeoff(altitude=50)

# ניווט לנקודה
await adapter.goto_position(lat=32.0853, lon=34.7812, alt=50)
```
