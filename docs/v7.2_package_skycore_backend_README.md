# SkyCore Pro - Backend API

## Overview

Backend API server for SkyCore Pro drone control system.

## Quick Start

```bash
cd backend
npm install
npm start
```

## Endpoints

### Authentication
- `POST /api/auth/login` - Login
- `POST /api/auth/register` - Register
- `GET /api/auth/verify` - Verify token

### Drones
- `GET /api/drones` - List all drones
- `GET /api/drones/:id` - Get drone details
- `POST /api/drones` - Register drone
- `PUT /api/drones/:id/status` - Update drone status

### Missions
- `GET /api/missions` - List missions
- `GET /api/missions/:id` - Get mission
- `POST /api/missions` - Create mission
- `PUT /api/missions/:id` - Update mission

### Telemetry
- `GET /api/telemetry/:droneId` - Get telemetry
- `POST /api/telemetry` - Submit telemetry

### Alerts & Logs
- `GET /api/alerts` - Get alerts
- `POST /api/alerts` - Create alert
- `GET /api/logs` - Get logs

## WebSocket

Connect to `ws://localhost:5760` for real-time telemetry.

### Message Types

```json
// Send command
{ "type": "COMMAND", "command": "ARM", "drone_id": "DRONE-1" }

// Receive telemetry
{ "type": "TELEMETRY_UPDATE", "lat": 32.0, "lon": 35.0, "alt": 50 }
```

## Environment Variables

```
PORT=3000
WS_PORT=5760
JWT_SECRET=your-secret-key
```
