## 1. הקדמה

### 1.1 אודות SkyCore Pro

**SkyCore Pro** היא מערכת בקרת רחפן מתקדמת הכוללת:

- **8 שכבות ארכיטקטורה** לבקרה אמינה
- **AUKF (Adaptive Unscented Kalman Filter)** לפילטרציה מתקדמת
- **ניווט אוטונומי** עם הימנעות ממכשולים
- **Geofencing** דינמי
- **Remote ID** עומד בתקנות FAA/EASA
- **הצפנה מ256-bit** לתקשורת מאובטחת
- **סימולציה מלאה** ב-SITL/Gazebo

### 1.2 דרישות מערכת

| רכיב | דרישה מינימלית | דרישה מומלצת |
|------|----------------|--------------|
| מעבד (CPU) | 4 ליבות | 8 ליבות |
| זיכרון (RAM) | 8 GB | 16 GB |
| אחסון | 50 GB SSD | 256 GB SSD |
| מערכת הפעלה | Ubuntu 20.04+ / Windows 10+ / macOS 12+ | Ubuntu 22.04+ |
| Python | 3.8+ | 3.10+ |
| חיבור רשת | Ethernet 100Mbps | Ethernet 1Gbps |

## 2. התקנה

### 2.1 התקנה מהירה (Quick Start)

```bash
# שכפול המאגר
git clone https://github.com/skycore/skycore-pro.git
cd skycore-pro

# הפעלת סקריפט התקנה
chmod +x install.sh
./install.sh

# הפעלת המערכת
python -m skycore.main --mode=gcs
```

### 2.2 התקנה עם Docker

```bash
# בניית תמונת Docker
docker build -t skycore-pro:latest .

# הרצת מסוף GCS
docker run -p 8000:8000 -p 14550:14550 skycore-pro:latest
```

### 2.3 התקנה ידנית

```bash
# יצירת סביבה וירטואלית
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
# או: venv\Scripts\activate  # Windows

# התקנת תלויות
pip install -r requirements.txt

# הגדרת משתני סביבה
export SKYCORE_CONFIG=./config/default.yaml
```

### 2.4 אימות התקנה

```bash
# בדיקת גרסה
python -m skycore --version

# בדיקת רכיבים
python -m skycore.cli health

# הפעלת בדיקה עצמית
python -m skycore.self_test
```

## 4. תכנון משימות

### 4.1 יצירת משימה חדשה

1. לחץ על **"📋 משימה חדשה"** בסרגל הכלים
2. בחר סוג משימה:
   - **Survey** - סקר שטח
   - **Inspection** - בדיקת תשתיות
   - **Delivery** - משלוח
   - **Custom** - משימה מותאמת אישית

### 4.2 הגדרת Waypoints

```python
# דוגמה: יצירת משימת סקר
from skycore.planning.mission import Mission, Waypoint

mission = Mission(name="Field Survey 2024")

# הוספת נקודות Waypoint
mission.add_waypoint(Waypoint(
    lat=32.0853,
    lon=34.7818,
    alt=100.0,
    action="survey",
    speed=10.0
))

mission.add_waypoint(Waypoint(
    lat=32.0860,
    lon=34.7825,
    alt=100.0,
    action="photo",
    speed=5.0
))

# שמירת המשימה
mission.save("field_survey_2024.json")
```

### 4.3 תכנון נתיב אוטומטי

```python
# תכנון נתיב עם הימנעות ממכשולים
from skycore.navigation.planner import PathPlanner

planner = PathPlanner()

# הגדרת מכשולים
obstacles = [
    {"lat": 32.0855, "lon": 34.7820, "radius": 20, "height": 80},
    {"lat": 32.0860, "lon": 34.7830, "radius": 15, "height": 60},
]

# תכנון נתיב
path = planner.plan_path(
    start=(32.0853, 34.7818),
    end=(32.0870, 34.7840),
    obstacles=obstacles,
    algorithm="rrt*"  # או "a_star", "dubins"
)
```

from skycore.swarm.coordinator import SwarmCoordinator

## 5. בקרת טיסה

### 5.1 מצבי טיסה (Flight Modes)

| מצב | תיאור | שימוש |
|-----|-------|-------|
| **MANUAL** | שליטה ידנית מלאה | אימון, טיסה חופשית |
| **STABILIZED** | ייצוב מופעל, ללא GPS | טיסה בתוך מבנה |
| **ALT_HOLD** | החזקת גובה אוטומטי | צילום, סקר |
| **POS_HOLD** | החזקת מיקום (GPS) | ריחוף במקום |
| **AUTO** | משימה אוטומטית | ביצוע משימה |
| **RTL** | חזרה לנקודת התחלה | חזרה בטוחה |
| **LAND** | נחיתה אוטומטית | נחיתה |
| **EMERGENCY** | עצירת חירום | מצב חירום |

### 5.2 פקודות טיסה בסיסיות

```python
from skycore.control.controller import DroneController

controller = DroneController()

# התחבר לרחפן
await controller.connect("udp:0.0.0.0:14550")

# המראה
await controller.arm()          # הפעלת מנועים
await controller.takeoff(50)    # המראה לגובה 50 מ'

# תנועה
await controller.set_position(32.0860, 34.7825, 60)  # X, Y, Z
await controller.set_velocity(5.0, 0, 0)             # vx, vy, vz

# סיבוב
await controller.set_yaw(90)   # סיבוב ל-90 מעלות

# נחיתה
await controller.land()

# ניתוק
await controller.disconnect()
```

### 5.3 פקודת E-Stop (עצירת חירום)

⚠️ **חשוב**: E-Stop מיידי מכבה את כל המנועים!

```python
from skycore.safety.estop import EmergencyStop

estop = EmergencyStop()

# עצירת חירום מיידית
await estop.trigger()

# ביטול נעילת E-Stop (דורש קוד אימות)
await estop.release(auth_code="RESET123")
```

## 6. מערכות בטיחות

### 6.1 Geofencing

Geofencing יוצר "גדרות וירטואליות" שהרחפן לא יכול לעבור.

#### סוגי גדרות:

| סוג | תיאור | שימוש |
|-----|-------|-------|
| **Cylinder** | גליל אנכי | הגבלת רדיוס טיסה |
| **Polygon** | מצולע | אזור טיסה מורכב |
| **3D Box** | קופסה תלת-ממדית | נפח טיסה מוגדר |
| **Altitude** | תקרה/רצפה | הגבלת גובה |

#### הגדרת Geofence:

```python
from skycore.safety.geofence import Geofence, FenceAction

# גדרת רדיוס
no_fly_zone = Geofence(
    name="Airport Restricted",
    fence_type="cylinder",
    center=(32.0853, 34.7818),
    radius=5000,  # מטרים
    min_alt=0,
    max_alt=120,
    action=FenceAction.RTL  # חזרה לנקודת התחלה
)

# הוספה למערכת
validator = GeofenceValidator()
validator.add_fence(no_fly_zone)
```

### 6.2 מערכת Failsafe

מערכת Failsafe מגנה על הרחפן במצבי חירום:

#### סוגי Failsafe:

| סוג | טריגר | פעולה |
|-----|-------|-------|
| **Battery Low** | < 20% | אזהרה → RTL |
| **Connection Lost** | > 10 שניות | Hover → RTL |
| **GPS Loss** | 0 satellites | Stabilized mode |
| **EKF Variance** | > 1.0 | Land immediately |
| **Geofence Breach** | Exit zone | RTL or Land |

#### הגדרת Failsafe:

```python
from skycore.safety.failsafe import FailsafeManager, FailsafeType

failsafe = FailsafeManager()

# הגדרת סף סוללה
failsafe.set_threshold(FailsafeType.BATTERY_WARNING, 30)   # %
failsafe.set_threshold(FailsafeType.BATTERY_CRITICAL, 15)  # %
failsafe.set_threshold(FailsafeType.BATTERY_EMERGENCY, 5)  # %

# הפעלת failsafe
await failsafe.start_monitoring()
```

### 6.3 Terrain Following

מערכת שמירה על גובה קבוע מעל פני השטח:

```python
from skycore.navigation.terrain import TerrainFollower, TerrainDatabase

# יצירת מערכת
terrain_db = TerrainDatabase()
follower = TerrainFollower(terrain_db)

# טעינת נתוני שטח
loader = TerrainDataLoader()
await loader.load_terrain_region(
    min_lat=32.0, min_lon=34.7,
    max_lat=32.2, max_lon=35.0,
    source="srtm"
)

# תכנון נתיב עם מעקב שטח
path = follower.plan_terrain_aware_path(
    start=(32.0853, 34.7818),
    end=(32.0900, 34.7900),
    desired_agl=50  # 50 מטר מעל השטח
)
```

### 6.4 Remote ID

שידור זיהוי רחפן בהתאם לתקנות:

```python
from skycore.communication.remote_id import RemoteIDModule

# יצירת מודול Remote ID
remote_id = RemoteIDModule(config={
    'id_type': 'serial',
    'id_value': 'SKY-123456',
    'operator_id': 'OP-789012',
    'protocol': 'mavlink'  # או 'wifi', 'ble'
})

# הפעלת שידור
await remote_id.start_broadcast()

# עדכון מיקום
remote_id.set_location(32.0853, 34.7818, 100)
remote_id.set_flying(True)

# בדיקת תאימות
from skycore.communication.remote_id import RemoteIDCompliance

compliance = RemoteIDCompliance()
result = compliance.check_compliance('faa', remote_id)

if result['compliant']:
    print("✓ עומד בתקנות FAA Part 107")
else:
    print(f"✗ כשלים: {result['failed_checks']}")
```

## 7. ניטור והתראות

### 7.1 הגדרת התראות

```python
from skycore.alerts import AlertManager, AlertChannel

alerts = AlertManager()

# הוספת ערוצי התראה
alerts.add_channel(AlertChannel.EMAIL, {
    'smtp_host': 'smtp.gmail.com',
    'smtp_port': 587,
    'from': 'drone@skycore.com',
    'to': ['pilot@company.com']
})

alerts.add_channel(AlertChannel.SMS, {
    'twilio_sid': 'YOUR_SID',
    'twilio_token': 'YOUR_TOKEN',
    'from': '+1234567890',
    'to': ['+0987654321']
})

alerts.add_channel(AlertChannel.WEBHOOK, {
    'url': 'https://api.company.com/webhooks/drone'
})

# הגדרת כללי התראה
alerts.add_rule('battery_low', {
    'condition': 'battery < 20',
    'severity': 'critical',
    'channels': ['email', 'sms', 'webhook']
})

alerts.add_rule('geofence_warning', {
    'condition': 'distance_to_fence < 100',
    'severity': 'warning',
    'channels': ['email']
})
```

## 9. פתרון בעיות

### 9.1 בעיות נפוצות ופתרונות

| בעיה | סיבה אפשרית | פתרון |
|------|-------------|-------|
| רחפן לא מתחבר | כבל/רשת | בדוק חיבור UDP, restart |
| GPS לא ננעל | מכשולים | עבור לאזור פתוח |
| סוללה מתחממת | עומס יתר | הפחת עומס, קירור |
| EKF divergence | חיישן פגום | כיול מחדש |
| משימה לא מתחילה | נתונים חסרים | בדוק waypoints |

### 9.2 מצבי שגיאה

#### Boot Failure
```
[FATAL] System boot failed
[ERROR] Component: IMU
[CAUSE] Sensor calibration timeout
```
**פתרון**: הפעל מחדש, בדוק כיול חיישנים

#### Communication Lost
```
[WARN] Heartbeat timeout: 5.0s
[INFO] Entering failsafe mode: LOITER
```
**פתרון**: בדוק חיבור רשת, הגדל טווח אנטנה

#### Geofence Breach
```
[CRITICAL] Geofence breach detected
[ACTION] Initiating RTL
[ESTIMATED_TIME] 45s to home
```
**פתרון**: בדוק הגדרות geofence, המתן לחזרה
