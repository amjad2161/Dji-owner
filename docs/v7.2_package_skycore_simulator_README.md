# SkyCore Pro - SITL Drone Simulator

Physics-based drone simulation system for GrokDrone Pro GCS.

## Features

- **Full Physics Simulation**: 6DOF drone dynamics with gravity, drag, thrust
- **Autopilot Modes**: Guided, Auto, RTL, Land, Loiter, AltHold, Stabilize, Acro
- **Real Telemetry**: 10Hz position, velocity, attitude, battery, GPS data
- **WebSocket Interface**: Real-time communication with GCS
- **MAVLink Protocol**: Industry-standard drone communication

## Quick Start

### 1. Start the Simulator

```bash
cd /workspace/skycore/simulator
python3 run_simulator.py
```

The simulator will start on `ws://0.0.0.0:5760`

### 2. Connect GCS

Open the GCS in your browser and click **Connect** (default: 127.0.0.1:5760)

### 3. Fly!

1. Click **Arm** to arm the drone
2. Click **Takeoff** or use the map to navigate
3. Click anywhere on the map to select a destination
4. Choose **Fly To** for immediate navigation or **Add Waypoint** for mission planning

## Flight Modes

| Mode | Description |
|------|-------------|
| **Stabilize** | Manual control with self-leveling |
| **Acro** | Manual rate control |
| **AltHold** | Altitude hold with manual control |
| **Guided** | Autonomous navigation to waypoints |
| **Auto** | Execute uploaded mission |
| **Loiter** | Hold position with GPS |
| **RTL** | Return to launch |
| **Land** | Automated landing |

## Telemetry Data

The simulator broadcasts:
- Position (lat/lon/alt)
- Velocity (NED frame)
- Attitude (roll/pitch/yaw)
- Battery status
- GPS info (satellites, HDOP, fix type)
- Flight mode and arming status

## Customization

### Change Simulation Parameters

Edit `mavlink_bridge.py`:

```python
# Physics parameters
self.mass = 1.5  # kg
self.drag_coefficient = 0.5
self.max_thrust = 30.0  # N per motor

# Home location
self.state.latitude = 31.7683  # Tel Aviv
self.state.longitude = 35.2137
```

### Wind Simulation

Add to `DroneSimulator.update()`:

```python
self.wind = Vector3(2.0, 0.5, 0.0)  # m/s N, E, D
```

## API Commands

| Command | Description | Parameters |
|---------|-------------|------------|
| ARM | Arm motors | param1: 1 |
| DISARM | Disarm motors | param1: 0 |
| TAKEOFF | Takeoff | param7: altitude |
| LAND | Land | - |
| RTL | Return to launch | - |
| FLY_TO | Navigate to point | lat, lon, alt |
| SET_MODE | Change flight mode | mode |
| UPLOAD_MISSION | Upload waypoints | waypoints[] |
| START_MISSION | Begin mission | - |

## Troubleshooting

### GCS won't connect

1. Check simulator is running: `ps aux | grep python`
2. Verify port 5760 is open: `netstat -tlnp | grep 5760`
3. Check firewall: `sudo ufw status`

### Drone not moving

1. Arm the drone first
2. Set flight mode to Guided or Auto
3. Send FLY_TO command or start mission

### Position stuck at home

Check the simulation loop is running:
```python
# Should see "Telemetry: lat=X, lon=Y" output
```
