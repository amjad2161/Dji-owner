# SkyCore - Autonomous Drone Platform

**Version 1.0.0** - A comprehensive autonomous drone system with 22-state AUKF navigation, multi-controller support, C-UAS detection, swarm coordination, and full GCS web interface.

## 🎯 Features

### Navigation (22-state Adaptive UKF)
- **Adaptive Unscented Kalman Filter** with LAMBDA RTK for sub-meter precision
- 45 sigma points for accurate state estimation
- 22 states: position (3), velocity (3), quaternion (4), gyro bias (3), accel bias (3), wind (3), clock bias (1), clock drift (1)
- **Target Performance**: <0.5m RTK accuracy, <2m GNSS accuracy, <0.5° attitude, <1ms runtime

### Control Systems (6 Controllers)
| Controller | Description |
|------------|-------------|
| **PID** | Standard, PI-D, I-PD variants for altitude/position/attitude/rate |
| **Geometric SE(3)** | Full attitude control on manifold |
| **LQR** | Optimal linear-quadratic regulation via Riccati equation |
| **MPC** | Linear, nonlinear, and convex optimization |
| **Motor Mixer** | Quad/Hex/Octo X/+ configurations |
| **Trajectory** | Minimum snap, minimum Jerk, emergency trajectory |

### Sensors (5 Sensors)
- **IMU** - Accelerometer/gyroscope with self-calibration
- **GNSS** - GPS with RTK support, satellite selection, ephemeris validation
- **Barometer** - Barometric altitude with temperature compensation
- **Magnetometer** - Compass with tilt compensation, disturbance detection
- **LIDAR** - Distance sensor with obstacle mapping

### Safety Systems
- **Safety Monitor** - Battery, altitude, GPS, RC, GCS, temperature, wind, sensors
- **Geofence Validator** - Polygon and cylinder constraints
- **Failsafe Manager** - Configurable responses to failures
- **Safety Levels**: OK → WARNING → CAUTION → CRITICAL → EMERGENCY
- **Automatic Responses**: RTL on battery low, hold on GPS lost, emergency land on critical

### Communications (4 Protocols)
- **MAVLink 2.0** - Full protocol implementation
- **ExpressLRS** - High-speed RC link support
- **Satellite** - Iridium/Globalstar uplink
- **MQTT** - Telemetry broker

### Perception
- **Obstacle Detection** - YOLO-based detection with NMS
- **Depth Estimation** - Stereo matching, LIDAR fusion, refinement

### Additional Features
| Feature | Description |
|---------|-------------|
| **C-UAS** | RF cross-correlation, acoustic detection, multi-sensor fusion (detection only, no attack) |
| **Swarm** | Formation control, task allocation, collision avoidance (up to 10 drones) |
| **Digital Twin** | Physics simulation with sensor latency modeling |
| **Voice Control** | Natural language commands with safety confirmation |
| **API Integration** | OpenSky Network, Weather API, OpenRouter AI |

## 🚀 Quick Start

### Installation
```bash
pip install numpy scipy
```

### Run the Desktop Application (Recommended)
```bash
python main.py --desktop
```
Opens a complete desktop GCS with real-time telemetry, charts, map, and mission control.

### Start CLI System
```bash
python main.py
```

### Start GCS Web Interface
```bash
python main.py --gcs --gcs-port 8000
```
Then open http://localhost:8000 in your browser.

### Run Tests
```bash
pytest tests/test_integration.py -v
```

### View System Info
```bash
python main.py --info
```

## 📋 State Machine

```
                    DISARMED ←────→ ARMED
                         │            │
                         ↓            ↓
                   [ARM_REQUEST]  [TAKEOFF_CMD]
                         │            │
                         ↓            ↓
                       TAKEOFF ────→ HOLD
                          │            │
                          ↓            ↓
                    +────-+-----+    AUTO
                    │           │      │
                    ↓           ↓      ↓
                 MANUAL     AUTO ──→ HOLD
                    │           │      │
                    ↓           ↓      ↓
                    +───→ RTL ←──+  LANDING
                    │       │        │
                    ↓       └───────+
                E_LAND
                    │
                    ↓
                ESTOP (terminal)
```

**States**: DISARMED, ARMED, TAKEOFF, HOLD, AUTO, MANUAL, RTL, LANDING, EMERGENCY_LANDING, FAILSAFE, ESTOP

**Triggers**: ARM_REQUEST, DISARM_REQUEST, TAKEOFF_CMD, MISSION_START, PAUSE, LAND_CMD, E_LAND_CMD, ESTOP_CMD, FAILSAFE_TRIGGER, GPS_LOST, GPS_REGAINED, BATTERY_LOW, BATTERY_CRITICAL, RC_LOST, RC_REGAINED, GEOFENCE_VIOLATION, MODE_SWITCH

## ⚖️ Safety Compliance

- Threat detection via RF cross-correlation and acoustic analysis
- Multi-sensor fusion for confidence scoring

- Alert system for operator awareness

## 🔧 API Reference

### Core System
```python
from skycore import SkyCoreSystem, create_system

# Create and initialize
system = create_system("config/default.json")
system.initialize()

# Control
system.arm()
system.takeoff(altitude=20)
system.land()
system.rtl()
system.emergency_stop()

# Mission
system.load_mission({"waypoints": [...]})
system.start_mission()
system.pause_mission()

# State
state = system.get_state()
```

### State Machine
```python
from skycore import FlightStateMachine, FlightState, TriggerEvent, SafetyLimits

sm = FlightStateMachine(safety_limits=SafetyLimits())
sm.trigger(TriggerEvent.ARM_REQUEST)
sm.trigger(TriggerEvent.TAKEOFF_CMD)
sm.trigger(TriggerEvent.MISSION_START)
```

### Safety Monitor
```python
from skycore import SafetyMonitor, SafetyConfig

sm = SafetyMonitor(config=SafetyConfig())
sm.update_state({"battery": {"percent": 25}})
violations = sm.check_all()
status = sm.get_status()
```
