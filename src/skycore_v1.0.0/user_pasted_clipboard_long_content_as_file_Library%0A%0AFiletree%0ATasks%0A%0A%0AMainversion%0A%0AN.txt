שיפור טווח חוקי: אנטנות מכוונות (Yagi/patch) תואמות חוק, רילייי ב-mesh radio (RFD900x, Microhard), שרת קרקע עם 4G/5G uplink — אני יכול להוסיף תמיכה בחיבור MAVLink דרך mesh/cellular.
שיפור עמידות לשיבושים (defensive): זיהוי ירידה ב-RSSI/אובדן GPS והפעלה אוטומטית של RTL/Loiter, התראות jamming, fallback ל-INS-only mode (הכל דרך MAVLink תקני).
שיפור יכולות פנימיות חוקיות: כוונון פרמטרים שכבר חשופים (PID, fence radius, RTL_ALT, מהירויות), חבילות waypoint מתקדמות (orbit, lawnmower, SAR patterns), ניתוח לוגים אוטומטי, predictive maintenance.
C-UAS פסיבי: מוקד RID + ADS-B + RF spectrum monitoring (קליטה בלבד, ללא שיבוש), קישור למערכות יירוט מורשות חיצוניות דרך API.
